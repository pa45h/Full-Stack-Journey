Introduction
  • The digital world is built on several key concepts—from the infrastructure of the Internet to the presentation of content on the World Wide Web.
  • In this guide, you’ll learn what the Internet is, how the Web functions, and how HTML forms the backbone of web development.
  • We’ll also provide a brief overview of the most commonly used HTML tags.

  1. The Internet vs. The World Wide Web

    2. The Internet
    • The Internet is a global network of computers that communicate using standardized protocols.
    • It’s the underlying infrastructure that enables data transfer, email, file sharing, and countless online services.
    The World Wide Web
    • The Web is a collection of interconnected documents and resources accessed through web browsers.
    • It uses the Internet to transmit data and relies on protocols like HTTP to deliver web pages to users.

  2. An Overview of Web Development

    3. What Is Web Development?
    • Web development involves creating and maintaining websites and web applications.
    • It combines content creation, design, and functionality.
    Core Technologies
    • HTML (HyperText Markup Language): Provides the structure for web pages.
    • CSS (Cascading Style Sheets): Controls the visual presentation of HTML elements.
    • JavaScript: Adds interactivity and dynamic behavior to websites.

  3. Introduction to HTML

    4.
    What Is HTML?
    • HTML is the standard markup language for creating web pages. It uses a series of tags to define the structure and content of a document.
    Basic Document Structure
    • An HTML document starts with a declaration (<!DOCTYPE html>), followed by the <html> element that encloses the <head> (for metadata and the title) and the <body> (which contains the content visible to users).

  4. Essential HTML Tags – A Quick Guide

    5. Structural Tags
    • <html> : The root element of an HTML document.
    • <head> : Contains metadata, such as the page title and links to scripts or stylesheets.
    • <body> : Encloses all visible content on the webpage.
    Semantic Tags
    • <header> and <footer>: Define the top and bottom sections of a page.
    • <nav> : Contains navigation links.
    • <main> : Wraps the main content of the page.
    • <section>, <article> and <aside> : Organize content into thematic groups.
    Content Tags
    • <h1> -- <h6> : Establish a hierarchy for the content.
    • <p> : Represents a paragraph of text.
    • <a> : Creates hyperlinks to other pages or resources.
    • <img> : Embeds images (with an alt attribute for accessibility).
    Grouping Tags
    • <div> : A generic block-level container for grouping elements.
    • <span> : A generic inline container for grouping text or inline elements.
    List and Table Tags
    • <ul> and <ol> : Create unordered (bullet) and ordered (numbered) lists.
    • <li> : Defines each list item.
    • <table>, <th>, <tr> and <td> : Used for displaying data in rows and columns.
    Form Tags
    • <form> : Encapsulates elements for user input.
    • <input type = "__"> :
      • text: For single-line text input.
      • password: Similar to text but hides the input (often replaced by dots or asterisks) to keep it secure.
      • email: Validates that the entered text follows an email address format.
      • number: Accepts only numerical input.
      • checkbox: Allows the user to select one or more options.
      • radio: Lets the user select only one option from a set.
      • file: Provides a button to browse and select files for upload.
      • hidden: Stores data that is not visible to the user but will be submitted with the form.

Conclusion:

  • The Internet and the World Wide Web form the foundation of our digital lives, and web development is the process by which we bring content and interactivity to these platforms.
  • Understanding HTML and its essential tags is the first step toward creating effective, accessible, and dynamic web pages.
